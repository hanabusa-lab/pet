機械学習コード説明

以下全て、ssd_kerasディレクトリ直下で実行。

●特徴抽出プログラム
-実行
　python ssd_test.py
-入力
　１ディレクトリに納められた画像ファイル群
　(data/images_minをハードコーディングで指定)
-出力
　画像ファイル群の特徴量テキスト
　(data/data.txt)
-説明
　画像ファイル群を読み込み、各々特徴抽出する。
　現状の特徴は下記の通り。
　1.画像でワンちゃんの占める割合
　2.ワンちゃんの認識スコア
　3.最大ワンちゃんのx方向中心度
　4.最大ワンちゃんのy方向中心度
-ライブラリ
 Kerasを用いてSSDのサンプルコードを用いている。
 参照ページ：http://app.road.jp.net/?p=1959

●画像評価プログラム
-実行
　python svm.py
-入力
　画像ファイル群の特徴量テキスト
　(data/000_data.txt, 001_data.txtをハードコーディングで指定)
-出力
　コンソールに認識精度を出力
-説明
　入力した特徴量をSVMで学習、識別。
　精度は10交差検証で判定。
　過学習で精度に寄与しないようなので、ハイパーパラメータはデフォルト設定。
　入力した特徴量のうち、ワンちゃん不在のものは学習・識別に含めない。
　悪画像と良画像のデータ偏りを減らすため、そもそも判定の余地もないため。
